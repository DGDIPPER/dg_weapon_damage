**dg_weapon_damage** is a lightweight and standalone FiveM script that allows you to easily modify the damage values of any GTA base weapon as well as addon weapons.  
With a simple and clean configuration file, server owners can fine-tune weapon balance without touching game files or complicated frameworks.
